# Compatibility Matrix

The compatibility page shows which SonicSaaS features are supported across different SonicOS versions and device models.

## Why Compatibility Matters

SonicWall firewalls run different versions of SonicOS (primarily v6 and v7), and the API capabilities vary significantly between versions. Some features are only available on newer firmware, while others work differently depending on the platform generation.

## What You Can See

- **Feature support by SonicOS version** — which features work on v6 vs v7
- **Support status** — Full, Partial, None, or Broken (known platform bugs)
- **Category filtering** — filter by feature category (firmware, security, VPN, etc.)
- **Device model considerations** — any model-specific limitations

## Feature Categories

The compatibility matrix covers features across all platform areas:

- Device management and polling
- Firmware management
- Configuration backup and restore
- Security services
- VPN (IPSec and SSL-VPN)
- Local user management
- Policy enforcement
- Network and WAN management
