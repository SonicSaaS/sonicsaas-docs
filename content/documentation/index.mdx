# Documentation

The documentation feature extracts comprehensive configuration data from your managed firewalls and makes it searchable and exportable.

## What Gets Documented

SonicSaaS can extract and organize a wide range of configuration data from each device:

- **Network configuration** — interfaces, IP addresses, VLANs, zones
- **User accounts** — local users, groups, permissions
- **VPN tunnels** — IPSec and SSL-VPN configurations
- **Security policies** — access rules, NAT policies, service objects
- **Licensing** — license keys, expiration dates, warranty status
- **Device details** — model, serial, firmware, uptime

## Use Cases

- **Client documentation** — generate comprehensive documentation for each client's firewall
- **Compliance evidence** — provide auditors with current configuration snapshots
- **Knowledge transfer** — document configurations for team onboarding or vendor handoff
- **Change tracking** — compare documentation snapshots over time

## Integration with ITGlue

When the [ITGlue integration](/integrations/itglue) is configured, documentation can be automatically synced to your ITGlue instance, keeping your documentation platform current without manual updates.

## Documentation Cleanup

The cleanup tool helps you manage stale or outdated documentation records, keeping your documentation library accurate and current.
