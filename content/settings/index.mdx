# Settings

The settings section lets you configure your SonicSaaS instance, manage your team, and control operational parameters.

## Settings Areas

| Section | What you configure |
| --- | --- |
| [General](/settings/general) | Team name, organization management, notification preferences |
| [Operational Settings](/settings/operational-settings) | Poll intervals, concurrency limits, timeouts, and other operational parameters |
| [Team Members](/settings/team) | Invite members, assign roles, manage access |
| [API Keys](/settings/api-keys) | Create and manage API keys for automation |
| [Schedules](/settings/schedules) | Configure polling, backup, and sync schedules |
| [Account Security](/settings/security) | Password changes, MFA enrollment |

## Who Can Access Settings

Settings access is governed by [roles and permissions](/security/roles-and-permissions):

- **Owners and Admins** can modify all settings
- **Operators** can view settings but have limited modification rights
- **Viewers and Auditors** have read-only access to settings
