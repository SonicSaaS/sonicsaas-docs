# ConnectWise Migration

The ConnectWise migration tool helps you transition from standalone organization management to ConnectWise-based client management.

## When to Use This

If you started with SonicSaaS before connecting ConnectWise, you may have organizations that need to be linked to ConnectWise companies. The migration tool automates this process.

## How It Works

1. **Review matches** — the tool suggests mappings between your SonicSaaS organizations and ConnectWise companies based on name similarity
2. **Confirm or adjust** — review the suggested mappings and correct any mismatches
3. **Apply** — the tool links your organizations to ConnectWise companies and migrates device assignments

## What Gets Migrated

- Organization-to-company mappings
- Device assignments to the correct ConnectWise company
- Historical data associations

## Safety

- The migration can be previewed before applying
- All changes are audited in the audit log
- Device data is not modified — only the organizational linkage changes
