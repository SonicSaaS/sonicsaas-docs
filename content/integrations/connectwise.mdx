# ConnectWise PSA

The ConnectWise PSA integration connects SonicSaaS to your ConnectWise Manage instance for client mapping and operational workflows.

## What It Provides

- **Client mapping** — link SonicSaaS organizations to ConnectWise companies
- **Ticket integration** — create and track service tickets for device issues
- **Asset sync** — keep ConnectWise configuration records updated with device data

## Setup

To configure the ConnectWise integration:

1. Navigate to **Integrations → ConnectWise**
2. Enter your ConnectWise PSA URL and API credentials (public key + private key)
3. Map ConnectWise companies to SonicSaaS organizations
4. Configure which data flows between the platforms
5. Save and test the connection

Your ConnectWise API credentials are encrypted at rest using AES-256-GCM.

## Client Mapping

The key value of the ConnectWise integration is linking your SonicSaaS organizations to ConnectWise companies. This enables:

- Automatic organization context when creating tickets
- Client-aware reporting and documentation
- Streamlined workflows between your PSA and firewall management
