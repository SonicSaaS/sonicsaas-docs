# Metrics

The metrics dashboard provides historical performance data for your managed firewalls.

## What You Can See

- **CPU utilization** — processor load over time for each device
- **Memory usage** — RAM consumption trends
- **Throughput** — network traffic volume through the firewall
- **Uptime** — device availability and reboot history

## Fleet-Level View

The metrics dashboard aggregates data across your fleet, letting you:

- Identify devices that are consistently overloaded
- Spot performance trends before they become problems
- Compare resource utilization across device models
- Plan capacity upgrades based on actual usage data

## Per-Device Metrics

For detailed metrics on a specific device, navigate to the device's metrics tab from the [Device Detail](/devices/device-detail) page. Per-device metrics include live charts and historical data.

## Data Collection

Metrics are collected during regular device polling. The polling interval determines the granularity of your metrics data — more frequent polling provides higher-resolution charts.
